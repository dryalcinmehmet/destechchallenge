    ::::::::::::::::::::::::::::::::Destech Makefile Commands ::::::::::::::::::::::::::::::::

   Type the commands below into the terminal. You can find examples for each other.

Examples:

  | down:  ## help for down
  |     --> For stopping and removing development containers, networks and volumes.
  |  >> make down


  | dev:  ## help for dev
  |     --> For run development containers, networks and volumes.
  |  >> make dev


  | prod:  ## help for prod
  |     --> For run deelopment containers, networks and volumes.
  |  >> make prod


  | release:  ## help for release
  |     --> For bump version.
  |  >> make release

  | install:  ## help for install
  |     --> For necessary ubuntu packages installing.
  |  >> make install
